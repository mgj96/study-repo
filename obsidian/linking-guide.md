# 노트 링크 가이드

## 이 레포의 링크 방식

GitHub 어필용이므로 **일반 마크다운 링크** 사용.  
GitHub에서 클릭 가능하고, Obsidian에서도 그대로 동작.

```markdown
# 같은 폴더
[모델별 비교](model-comparison.md)

# 다른 폴더
[코드베이스 분석](../llm/codebase-analysis.md)

# 외부 링크
[nanoGPT](https://github.com/karpathy/nanoGPT)
```

## Obsidian에서 열기

1. Obsidian 설치 후 `study-repo` 폴더를 Vault로 열기
2. 설정 → Files & Links → "Use [[Wikilinks]]" 끄기
3. → 마크다운 링크 방식으로 통일
